<h1 title = "inline css" > Project on inline css </h1> 
<p> <mark> <em>  Inline CSS:</em> </mark> It is used to apply a unique style to a single  <abbr title="HTML (Hyper Text Markup Language)"> HTML </abbr> element. An inline CSS uses the style attribute of an HTML element. </p>
<h2 title = "In HTML"> About Internal CSS </h2>
<p> An <strong> Internal CSS:</strong> It is used to define the style for <del>MANY</del> SINGLE <abbr title="HTML (Hyper Text Markup Language)"> HTML</abbr> page. <br> It is defined in the <b><q> HEAD</q> </b> section of <abbr title="HTML (Hyper Text Markup Language)"> HTML</abbr> page. An internal CSS is defined in the <head> section of an HTML page, within a <style> element. </p>
<h3 title = "In HTML"> About External CSS </h3>
<p>  An external CSS is used to define the style for many <abbr title="HTML (Hyper Text Markup Language)"> HTML</abbr> pages. With an external style sheet, you can change the look of an entire web site, by changing one file!</p>
